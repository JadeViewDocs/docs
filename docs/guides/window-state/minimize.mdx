# 最小化窗口

## minimize_window

最小化指定窗口。

### 函数签名

```c
int minimize_window(uint32_t window_id);
```

### 参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| `window_id` | uint32_t | 窗口 ID |

### 返回值说明

| 返回值 | 说明 |
|---------|------|
| `1` | 成功 |
| `0` | 失败 |

### 功能描述

最小化指定窗口。

### 使用示例

```c
// 最小化 ID 为 1 的窗口
int result = minimize_window(1);

if (result == 1) {
    printf("窗口最小化成功\n");
} else {
    printf("窗口最小化失败\n");
}
```
