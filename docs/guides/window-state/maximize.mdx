# 切换窗口最大化/还原状态

## toggle_maximize_window

切换指定窗口的最大化/还原状态。

### 函数签名

```c
int toggle_maximize_window(uint32_t window_id);
```

### 参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| `window_id` | uint32_t | 窗口 ID |

### 返回值说明

| 返回值 | 说明 |
|---------|------|
| `1` | 成功 |
| `0` | 失败 |

### 功能描述

切换指定窗口的最大化/还原状态。如果窗口当前是最大化状态，则还原为正常大小；如果窗口当前是正常大小，则最大化窗口。

### 使用示例

```c
// 切换 ID 为 1 的窗口最大化/还原状态
int result = toggle_maximize_window(1);

if (result == 1) {
    printf("窗口最大化/还原状态切换成功\n");
} else {
    printf("窗口最大化/还原状态切换失败\n");
}
```
