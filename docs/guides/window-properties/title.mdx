# 设置窗口标题

## set_window_title

设置指定窗口的标题。

### 函数签名

```c
int set_window_title(uint32_t window_id, const char* title);
```

### 参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| `window_id` | uint32_t | 窗口 ID |
| `title` | const char* | 新的窗口标题 |

### 返回值说明

| 返回值 | 说明 |
|---------|------|
| `1` | 成功 |
| `0` | 失败 |

### 功能描述

设置指定窗口的标题。

### 使用示例

```c
// 设置 ID 为 1 的窗口标题
int result = set_window_title(1, "新窗口标题");

if (result == 1) {
    printf("窗口标题设置成功\n");
} else {
    printf("窗口标题设置失败\n");
}
```
