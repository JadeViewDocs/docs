# 设置窗口大小

## set_window_size

设置指定窗口的大小。

### 函数签名

```c
int set_window_size(uint32_t window_id, double width, double height);
```

### 参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| `window_id` | uint32_t | 窗口 ID |
| `width` | double | 新的宽度（像素） |
| `height` | double | 新的高度（像素） |

### 返回值说明

| 返回值 | 说明 |
|---------|------|
| `1` | 成功 |
| `0` | 失败 |

### 功能描述

设置指定窗口的大小。

### 使用示例

```c
// 设置 ID 为 1 的窗口大小为 1024x768
int result = set_window_size(1, 1024, 768);

if (result == 1) {
    printf("窗口大小设置成功\n");
} else {
    printf("窗口大小设置失败\n");
}
```
