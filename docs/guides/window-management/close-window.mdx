# 关闭窗口

## close_window

关闭指定 ID 的窗口，释放窗口资源。

### 函数签名

```c
int close_window(uint32_t window_id);
```

### 参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| `window_id` | uint32_t | 要关闭的窗口 ID |

### 返回值说明

| 返回值 | 说明 |
|---------|------|
| `1` | 成功 |
| `0` | 失败 |

### 功能描述

关闭指定 ID 的窗口，释放窗口资源。

### 使用示例

```c
// 关闭 ID 为 1 的窗口
int result = close_window(1);

if (result == 1) {
    printf("窗口关闭成功\n");
} else {
    printf("窗口关闭失败\n");
}
```

## request_redraw

请求指定窗口重绘，刷新窗口内容。

### 函数签名

```c
int request_redraw(uint32_t window_id);
```

### 参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| `window_id` | uint32_t | 要重绘的窗口 ID |

### 返回值说明

| 返回值 | 说明 |
|---------|------|
| `1` | 成功 |
| `0` | 失败 |

### 功能描述

请求指定窗口重绘，刷新窗口内容。

### 使用示例

```c
// 请求 ID 为 1 的窗口重绘
int result = request_redraw(1);

if (result == 1) {
    printf("窗口重绘请求成功\n");
} else {
    printf("窗口重绘请求失败\n");
}
```
