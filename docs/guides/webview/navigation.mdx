# 导航到 URL

## navigate_to_url

让指定窗口的 WebView 导航到指定的 URL。

### 函数签名

```c
int navigate_to_url(uint32_t window_id, const char* url);
```

### 参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| `window_id` | uint32_t | 窗口 ID |
| `url` | const char* | 要导航到的 URL |

### 返回值说明

| 返回值 | 说明 |
|---------|------|
| `1` | 成功 |
| `0` | 失败 |

### 功能描述

让指定窗口的 WebView 导航到指定的 URL。

### 使用示例

```c
// 让 ID 为 1 的窗口导航到 Google
int result = navigate_to_url(1, "https://www.google.com");

if (result == 1) {
    printf("导航成功\n");
} else {
    printf("导航失败\n");
}
```

## reload

重新加载指定窗口的 WebView 内容。

### 函数签名

```c
int reload(uint32_t window_id);
```

### 参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| `window_id` | uint32_t | 窗口 ID |

### 返回值说明

| 返回值 | 说明 |
|---------|------|
| `1` | 成功 |
| `0` | 失败 |

### 功能描述

重新加载指定窗口的 WebView 内容，相当于浏览器的刷新功能。

### 支持版本

0.2 以上

### 使用示例

```c
// 刷新 ID 为 1 的窗口的 WebView 内容
int result = reload(1);

if (result == 1) {
    printf("刷新成功\n");
} else {
    printf("刷新失败\n");
}
```
