# 设置窗口主题

## set_window_theme

设置指定窗口的主题。

### 函数签名

```c
int set_window_theme(uint32_t window_id, const char* theme);
```

### 参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| `window_id` | uint32_t | 窗口 ID |
| `theme` | const char* | 主题名称，可选值：`"Light"`、`"Dark"`、`"System"` |

### 返回值说明

| 返回值 | 说明 |
|---------|------|
| `1` | 成功 |
| `0` | 失败 |

### 功能描述

设置指定窗口的主题。

### 使用示例

```c
// 设置 ID 为 1 的窗口使用深色主题
set_window_theme(1, "Dark");

// 设置 ID 为 1 的窗口跟随系统主题
set_window_theme(1, "System");
```

## get_window_theme

获取指定窗口的当前主题。

### 函数签名

```c
int get_window_theme(uint32_t window_id);
```

### 参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| `window_id` | uint32_t | 窗口 ID |

### 返回值说明

| 返回值 | 说明 |
|---------|------|
| `0` | 浅色主题 |
| `1` | 深色主题 |

### 功能描述

获取指定窗口的当前主题。

### 使用示例

```c
// 获取 ID 为 1 的窗口主题
int theme = get_window_theme(1);

if (theme == 0) {
    printf("窗口当前使用浅色主题\n");
} else {
    printf("窗口当前使用深色主题\n");
}
```
