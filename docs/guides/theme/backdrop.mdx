# 设置窗口背景材料

## set_window_backdrop

设置指定窗口的背景材料效果。

### 函数签名

```c
int set_window_backdrop(uint32_t window_id, const char* backdrop_type);
```

### 参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| `window_id` | uint32_t | 窗口 ID |
| `backdrop_type` | const char* | 背景材料类型，可选值：`"mica"`、`"micaalt"`、`"acrylic"` |

### 返回值说明

| 返回值 | 说明 |
|---------|------|
| `1` | 成功 |
| `0` | 失败 |

### 功能描述

设置指定窗口的背景材料效果。

### 使用示例

```c
// 设置 ID 为 1 的窗口使用 Mica 背景效果
set_window_backdrop(1, "mica");

// 设置 ID 为 1 的窗口使用 Acrylic 背景效果
set_window_backdrop(1, "acrylic");
```

### 背景材料类型说明

- **mica**：Mica 效果，Windows 11 的默认背景材料
- **micaalt**：Mica Alt 效果，Mica 的替代版本
- **acrylic**：Acrylic 效果，半透明模糊背景
