# 易语言SDK

JadeView 易语言 SDK 是一个专为易语言开发者设计的 WebView 窗口库，基于 Rust 开发，提供了简单易用的 API 接口，帮助开发者快速构建现代化的桌面应用程序。

## 下载链接

- [GitHub Releases](https://github.com/JadeViewDocs/library/releases) - 官方发布渠道，包含最新版本的动态库和易语言 SDK 源码
- [精易论坛](https://bbs.125.la/) - 易语言开发者社区，可获取相关资源和支持

## 主要特性

- 基于最新的 WebView 技术，支持现代 Web 标准
- 简单易用的 API 设计，适合易语言开发者
- 支持多种窗口主题（亮色、暗色、自动）
- 灵活的窗口配置选项
- 支持透明窗口和自定义背景色
- 支持 JavaScript 与易语言的交互
