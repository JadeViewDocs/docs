# Web Core SDK

Web Core SDK 即将推出！

Web Core SDK 将提供 JavaScript/TypeScript 的完整 API 接口，方便开发者在前端项目中集成 JadeView。

## 即将推出的功能

- 完整的 JavaScript/TypeScript API
- 类型定义
- 事件监听和处理
- 窗口管理
- WebView 控制
- 主题管理
- IPC 通信

## 快速开始

```javascript
// 即将推出
import { JadeView } from '@jadeview/web-core';

const jadeview = new JadeView();
await jadeview.init();
```
