# 更新日志

## 版本历史

### 1.0.0 (开发中)

**新增功能：**
- 初始版本发布
- 完整的 C API 接口
- WebView 窗口管理
- 事件系统
- 主题管理
- IPC 通信
- 本地服务器支持

**核心功能：**
- 基于 Rust 开发的 WebView 窗口库
- C 语言兼容的 API 接口
- 跨平台支持（Windows 10/11）
- 基于 wry 库的 WebView 集成
- 灵活的窗口管理
- 丰富的事件类型和回调机制
- Light、Dark、System 主题切换
- 内置本地服务器功能
- 严格的内存管理机制

**技术栈：**
- Rust 标准库
- tao：窗口管理库
- wry：WebView 库

**文档：**
- 完整的 API 文档
- 快速开始指南
- 常见问题解答
- 代码示例
