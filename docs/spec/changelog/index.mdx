# 更新日志

## 版本历史

### v0.2.0 (2026-01-10)

[查看发布页面](https://github.com/JadeViewDocs/library/releases/tag/v0.2.0)

<div className="changelog-version-container">

**重要更新**
本次更新对 WebView 事件处理和窗口管理架构进行了重要改进，引入了更高效的事件通信模型。

**功能增强**
- **`webview-will-navigate` 事件增强**  
  为该事件添加���阻止导航的能力，允许应用在页面跳转前进行拦截和控制。
  
- **新增 C API 函数**  
  添加 `reload_webview_window` 函数，支持通过 C API 刷新 WebView 页面内容。

**架构优化**
- **事件通信模型重构**  
  将原先基于传统的轮询模型升级为唤醒的组合模式，实现更高效的事件传递。
- - 现在程序稳定性大幅度提升
- - 顺便修复了窗口启动偶尔掉速的情况

#### 性能提升
- **减少锁竞争**  
  新模型显著降低了多线程环境下的锁竞争，提高了系统整体响应速度。
  
- **支持批量处理**  
  优化后的事件循环支持批量处理消息，减少上下文切换开销。
  
- **逻辑线性化**  
  通过 Channel 通信机制使事件处理逻辑更加线性，降低了死锁风险。
  
- **高频事件处理能力**  
  新的架构能够更高效地处理高频事件，提升了在高负载场景下的稳定性。

###API 变更
- **`WindowManager` 新增方法**  
  在 `WindowManager` 中添加了 `reload` 方法，用于刷新管理的 WebView 窗口。

</div>

### v0.2.1 (2026-01-12)
<div className="changelog-version-container">

**功能增强**
- **新增 C API 函数**
  - 添加 `set_window_fullscreen` 函数，用于设置窗口全屏状态
  - 添加 `is_window_maximized` 函数，用于获取窗口是否最大化

- **WebView 窗口选项增强**
  - 在 `WebViewWindowOptions` 结构体中新增 `borderless` 字段，支持无边框窗口设置
  - 无边框选项：1 为无边框，去除边框和系统阴影

**API 变更**
- **移除 deprecated 函数**
  - 移除 `set_window_event_handlers` 函数，建议使用 `jade_on` 函数替代

**Bug 修复**
- 修复窗口启动偶尔掉速的情况
- 修复窗口大小调整时的卡顿问题

</div>
