# 示例代码

以下为可直接套用的代码片段，覆盖窗口 / IPC / 托盘 / 热键 / 右键菜单 / 对话框 / 拖放 / YAML 等能力。片段默认处于 `类 启动类 <公开 基础类 = 程序类>` 内，已引用 JadeView 模块并声明：

```wsv
变量 Jade <类型 = JadeView>
变量 主窗口ID <类型 = 整数>
```

> 接收事件方法（`JadeView_应用就绪` 等）可点击 `Jade` 变量旁的闪电图标快速生成，无需手写签名。

## IPC 应答与解析 JSON payload

```wsv
方法 IPC频道消息 <公开 类型 = 文本型 @强制输出 = 真 @输出名 = "ipcChannelMessageCallback">
参数 通道名 <类型 = 文本型>
参数 窗口ID <类型 = 整数>
参数 数据 <类型 = 文本型>
{
    变量 数据解构 <类型 = json_object>
    数据解构.创建自文本 (数据)

    如果 (通道名 == "ipc-echo")
    {
        返回 (数据)    // 原样回声
    }
    如果 (通道名 == "set-theme")
    {
        // 前端 payload: {"mode": "Dark"}
        Jade.窗口.置主题 (窗口ID, 数据解构.取文本 ("mode", "System"))
        返回 ("主题已切换")
    }
    如果 (通道名 == "minimize")
    {
        Jade.窗口.最小化 (窗口ID)
        返回 ("已最小化")
    }
    返回 ("未知通道: " + 通道名)
}
```

## 文件拖放：转发前端 + enter 同步拦截

```wsv
变量 拒绝拖入 <类型 = 逻辑型>

方法 JadeView_文件拖放 <接收事件 类型 = 整数 注释 = "需手动订阅；enter/drop 阶段可拦截。">
参数 来源对象 <类型 = JadeView>
参数 标记值 <类型 = 整数>
参数 窗口ID <类型 = 整数>
参数 数据 <类型 = 文本型>
{
    如果 (来源对象 == Jade)
    {
        // enter/over/drop/leave 全程转发回页面（宿主接管后页面收不到原生 drop）
        Jade.发送IPC消息 (窗口ID, "drag-drop", 数据)

        如果真 (拒绝拖入)
        {
            变量 拖放数据 <类型 = json_object>
            如果真 (拖放数据.创建自文本 (数据))
            {
                如果真 (拖放数据.取文本 ("type", "") == "enter")
                {
                    返回 (1)   // enter 阶段拒绝：光标变禁止，后续不会有 drop
                }
            }
        }
    }
    返回 (0)
}
```
