# API 参考

本页用于总览当前 JadeView 火山模块的主要能力。

## 入口类

- `JadeView`
- `JadeView_托盘`

## 常量类

- `主题颜色`
- `背景材质`
- `边框风格`
- `热键功能键`
- `托盘菜单项类型`
- `托盘事件类型`
- `路径常量`
- `对话框属性`
- `提示框类型`
- `订阅_窗口事件`
- `订阅_视图事件`
- `订阅_通知事件`
- `订阅_系统事件`

## 数据类型

- `JadeView_窗口选项`
- `JadeView_视图选项`
- `JadeView_通知参数`

## 主要功能分类

### 生命周期

- `初始化(...)`
- `运行消息循环()`
- `清理全部窗口()`

### 窗口与视图

- 创建窗口
- 导航页面
- 执行脚本
- 窗口状态控制
- 主题与材质控制

### 事件

- 自动订阅普通事件
- 手动订阅特殊事件
- 可拦截事件返回 `1` 表示拦截

### IPC

- `置IPC频道回调(...)`
- `注册IPC通道(...)`
- `注销IPC通道(...)`

### 对话框与通知

- 同步文件对话框
- 异步文件对话框
- 消息框
- 错误框
- 系统通知

### 托盘

- 创建托盘图标
- 添加菜单
- 处理托盘事件
- 处理托盘菜单命令事件

## 阅读建议

建议按这个顺序阅读：

1. `main.wsv`
2. `快速开始`
3. `JadeView 类`
4. `JadeView_托盘 类`
5. `高级用法`
