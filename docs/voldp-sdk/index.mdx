# JadeView 火山 SDK

JadeView 火山 SDK 是一个面向火山视窗项目的模块封装，用于在火山程序中直接调用 JadeView 提供的窗口、WebView、IPC、托盘、通知、对话框与系统集成功能。

## 模块安装与使用

使用方式很简单：

1. 下载 JadeView 火山模块安装包
2. 在火山环境中安装模块
3. 在项目中引用 `火山.JadeView` 包
4. 按文档调用类、方法、事件和常量

使用模块时，无需关注模块内部引用的：

- `lib`
- `cpp`
- 头文件
- 桥接层实现

## 模块能力

- 初始化 JadeView 运行时并启动消息循环
- 创建普通窗口与无边框窗口
- 导航页面、执行脚本、发送页面消息、处理 WebView 事件
- 订阅窗口、视图、通知、系统与托盘事件
- 注册和处理 IPC 通道
- 显示系统通知、文件对话框、消息框、错误框
- 注册全局热键、文件关联、自定义协议
- 管理托盘图标与托盘菜单

## 推荐示例工程

当前推荐优先阅读的完整示例工程是 `main.wsv`。

这份示例工程体现了当前模块的推荐用法：

- 在 `启动方法` 中先设置 IPC 回调，再注册频道，再补充特殊事件订阅
- 在 `初始化()` 成功后，通过 `应用就绪` 事件创建主窗口
- IPC 采用“对象 + 输出名”方式绑定回调
- 托盘通过 `JadeView_托盘` 独立管理，并在事件中处理菜单命令
- 通知、托盘、窗口与视图事件都在同一工程内串联验证
