# JadeView_托盘 类

`JadeView_托盘` 用于管理托盘图标、托盘菜单和托盘相关事件。

## 创建与销毁

### 创建

- 创建托盘图标
- 自动订阅：
  - `tray-event`
  - `tray-menu-command`
- 避免重复创建和重复订阅

### 销毁

- 自动注销托盘事件
- 销毁托盘资源

## 常用方法

- `置可见(...)`
- `置提示(...)`
- `置图标(...)`
- `添加菜单项(...)`
- `置菜单禁止(...)`
- `置菜单危险(...)`
- `置菜单标题(...)`
- `删除菜单(...)`
- `清空菜单()`

## 托盘交互事件

回调形态：

- `托盘交互事件(事件类型, 托盘ID)`

模块已从原始 `tray-event` JSON 中提取：

- `event`
- `tray_id`

## 托盘菜单命令事件

回调形态：

- `托盘菜单命令事件(托盘ID, 键名, 项目ID, 项目类型, 已选中, 风险项目)`

模块已从原始 `tray-menu-command` JSON 中提取：

- `tray_id`
- `key`
- `item_id`
- `item_type`
- `checked`
- `dangerous`

使用者无需自己再解析原始 JSON。
