# 快速开始

## 安装

1. 下载 JadeView 火山模块安装包
2. 在火山环境中安装模块
3. 在项目中引用 `火山.JadeView` 包

## 推荐先看什么

如果您希望先看一份已经跑通且覆盖完整能力的示例，优先参考 `main.wsv`。

它已经覆盖了：

- 初始化与消息循环
- IPC 回调与通道注册
- 手动订阅可拦截事件
- 应用就绪后创建主窗口
- 托盘创建、菜单添加、通知显示

## 推荐生命周期

当前推荐按这份示例工程的生命周期组织代码：

1. 在 `启动方法` 中：
   - `置IPC频道回调(...)`
   - `注册IPC通道(...)`
   - `订阅事件(...)`
   - `初始化(...)`
2. 在 `应用就绪` 事件中：
   - 配置窗口选项和视图选项
   - 创建主窗口
   - 初始化托盘和通知
3. 在各事件方法中：
   - 处理窗口、视图、托盘、通知和 IPC 回调

## 最小示例

```wsv
<火山程序 类型 = "通常" 版本 = 1 />

包 火山.样例

类 启动类 <公开 基础类 = 窗口程序类>
{
    方法 启动方法 <公开 类型 = 整数>
    {
        变量 Jade <类型 = 火山.JadeView.JadeView>

        如果 (Jade.初始化 () = 假)
        {
            调试输出 ("JadeView 初始化失败")
            返回 (0)
        }

        变量 窗口选项 <类型 = 火山.JadeView.JadeView_窗口选项>
        窗口选项.标题 = "JadeView 火山示例"
        窗口选项.宽度 = 1200
        窗口选项.高度 = 800

        变量 视图选项 <类型 = 火山.JadeView.JadeView_视图选项>
        变量 窗口ID <类型 = 整数>
        窗口ID = Jade.创建窗口 ("https://jade.run", 0, 窗口选项, 视图选项)

        如果 (窗口ID <= 0)
        {
            调试输出 ("创建窗口失败")
            返回 (0)
        }

        返回 (Jade.运行消息循环 ())
    }
}
```

## IPC 推荐写法

当前推荐采用“对象 + 输出名”方式绑定 IPC 回调：

```wsv
Jade.置IPC频道回调 (本对象, "ipcChannelMessageCallback")
Jade.注册IPC通道 ("windowAction")
Jade.注册IPC通道 ("setTheme")
```

对应的方法模板：

```wsv
方法 IPC频道消息 <公开 类型 = 文本型 @强制输出 = 真 @输出名 = "ipcChannelMessageCallback">
参数 通道名 <类型 = 文本型>
参数 窗口ID <类型 = 整数>
参数 数据 <类型 = 文本型>
{
    返回 ("")
}
```

## 非静态回调说明

以下场景推荐直接用对象方法，而不是静态回调：

- IPC 频道消息回调
- 异步对话框回调

推荐原因：

- 更符合火山项目的对象组织方式
- 方便直接访问本对象状态
- 避免把业务逻辑堆到全局静态方法里

## 事件处理

`JadeView.初始化()` 默认会自动订阅大部分非拦截事件。您只需要在对象上实现对应事件即可。

```wsv
方法 应用就绪 <公开 类型 = 整数>
参数 数据 <类型 = 文本型>
{
    调试输出 ("应用就绪", 数据)
    返回 (0)
}
```

## 可拦截事件

以下事件默认不会自动订阅，您应显式调用 `订阅事件(...)`：

- `订阅_窗口事件.窗口即将关闭`
- `订阅_视图事件.即将导航`
- `订阅_视图事件.请求新窗口`
- `订阅_视图事件.下载开始`
- `订阅_视图事件.文件拖入`

前四个可拦截事件的规则是：

- 返回 `1`：拦截
- 返回 `0`：放行

## 对话框回调

同步对话框直接返回 `文本型 JSON`。

异步对话框推荐用对象方法承接：

```wsv
方法 打开文件完成 <公开 类型 = 整数>
参数 结果JSON <类型 = 文本型>
{
    调试输出 (结果JSON)
    返回 (0)
}
```

```wsv
Jade.显示打开文件对话框_异步 (0, "打开文件", "", "", "", "", this, 打开文件完成)
```
